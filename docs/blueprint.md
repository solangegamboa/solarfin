# **App Name**: SolarFin

## Core Features:

- Dashboard Overview: User-friendly dashboard displaying current balance, recent transactions, and spending insights.
- Expense Tracking: Categorize expenses to track spending habits effectively. 
- Credit Card Management: Record and manage credit card purchases with installment plans, including interest calculations and payment reminders.
- Loan Management: Input and manage loan details, track payment schedules, and monitor outstanding balances.
- Recurring Expense Forecast: Add recurring expenses to automatically generate future expense forecasts for up to 6 months, aiding budget planning.
- AI Savings Suggestions: A financial planning tool providing personalized suggestions on savings based on a generative AI analysis of income and spending data. It recommends how to optimize investments or manage debt based on the specific user profile.
- User Authentication: Authentication system integrated with Firebase Authentication using email/password. Persisted to Data Connect on Firebase. Support for multiple clients.

## Style Guidelines:

- Primary color: Soft teal (#76D275) for a calming and trustworthy feel.
- Background color: Light green (#E9F5E8) for a clean, positive interface.
- Accent color: Warm gold (#D4AF37) for highlighting important information and actions.
- Body and headline font: 'PT Sans' for a balance of modern aesthetic with warmth.
- Use simple, clean icons to represent expense categories and financial functions.
- A clean, well-spaced layout with intuitive navigation and clear data visualization.
- Subtle transitions and animations for a smooth user experience.
